>The **delete** element is a stand-alone element that can be used in different contexts.
 On its own, it's a simple circle with a cross:
```jsx
<div>
  <Delete/>
</div>
```

>It comes in **4** sizes:

```jsx
<div>
  <Delete size="small"/>
  <Delete/>
  <Delete size="medium"/>
  <Delete size="large"/>
</div>
```
