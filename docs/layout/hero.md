# Hero Components

Heros are a great way to feature information and titles in a colorful way within your application. 

```jsx
<Hero color="success" bold>
  <Hero.Body>
  <Container>
    <Title>
      Hero Title
      <Subtitle>
        Hero Subtitle
      </Subtitle>
    </Title>
  </Container>
  </Hero.Body>
</Hero>
```
